What is 'scope', and how does it work in JavaScript?
In Javascript we work a lot with variables; some things are determined/defined by how it relates to something else.
E.g. I want to get the value of "This" but to do that I need the input of "That".
There is an overall scope in our code (Global) but also, whenever we create a new function in JS, we create a new scope (local).
Scope determines the accessibility of variables. Things operating in the global scope are accessable to each other. Things within a local scope are accessable to the other things in that local scope, all the scopes can see/access the global scope..
By haveing functions within functions you create local scopes within local scopes (Nested Scope).
What does all this mean?
Say I have written some code, it has many variables within it. I have one function within my code (function A). This function contains variables and one other function (function B) with it's ow variables.
The Function A has local scope. It's variables can access the global scope but the global scope may not be able to access Function A's variables.
Function B also has a local scope. It's variables can access Function A's variables and the global variables, however Function A may not be able to access Function B's variables.
Each local scope can see all the local scopes that can contain it and all scopes can see the global scope. This is called "lexical scoping".
